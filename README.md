# Markdown to PDF Converter

Free online tool to convert Markdown to PDF. Works with ChatGPT outputs, resumes, and docs.

## Features

- Real-time preview
- One-click PDF export
- No signup required
- Works on mobile

## Usage

1. Paste Markdown
2. Preview
3. Export as PDF

## Local Setup

```bash
git clone https://github.com/maksimkunaev/markdown_to_pdf.git
cd markdown_to_pdf
open index.html
```
